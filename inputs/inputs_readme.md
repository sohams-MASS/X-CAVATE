# Inputs Folder

The inputs folder will contain the network files (output from SimVascular).

## Inputs: Files

The inputs folder contains two required files to define the network, which are the output from SimVascular:
1. .txt file defining the network as b-splines
2. .txt file identifying the inlet and outlet nodes

The inputs folder also contains two optional files, for computing Q, generated by calibration.py and calibration_validation.py:
1. diameters.csv
2. diameters_validation.csv

## Inputs: Subfolders

The inputs folder contains two subfolders:
1. inputs/custom
2. inputs/diameters
